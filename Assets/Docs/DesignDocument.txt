НАЗВАНИЕ:
	Whac a Mole... and More

ОБЩЕЕ ОПИСАНИЕ:
	Развитие классической аркадной игры Whac a Mole. Версия для смартфонов.

ЦЕЛЬ ИГРЫ:
	- Набрать как можно больше монет на каждом уровне, ударяя существ вылезающих из нор.
	- Для каждого профиля сохраняются результаты прохождения каждого уровня (время, число монет) в TopScores.

ПРАВИЛА:
	- В начале игры игрок имеет 0 монет и 10 жизней.
	- Уровень заканчивается, когда кончаются жизни.
	- Для перехода на следующий уровень необходимо набрать заданный для уровня минимум монет.
	- Пустую нору можно "закопать" ударив по ней (как это будет влиять на геймплей?)
	- Игрок может зарабатывать/терять монеты, жизни и способности ударяя существ появляющихся из нор или же наоборот, не успевая их ударить до того, как они исчезнут.

УРОВНИ:
	Каждый уровень имеет набор базовых настроек, отличающих один уровень от другого. Начальные уровни простые (по сложности и наполнению). Последующие уровни усложняются и имеют больше геймплейных возможностей.
НАСТРОЙКА УРОВНЕЙ:
	- Тема оформления: 1. Набор тайлов из которых случайным образом собирается фон. 2. Набор тайлов для нор. Темы добавляют визуального разнообразия уровням и не влияют на геймплей.
	- Настройки HolesCreator (создание случайных нор): 1. Размер нор, 2. Минимальное расстояние между норами, 3. Минимальное расстояние от краев экрана. Можно попробовать реализовать размещение нор по сетке с заданными харакеристиками.
	- Настройки Generator (скорость и процесс игры): 1. Частота появления существ из нор, 2. Время пребывания существа в норе (потом исчезает), 3. Ускорение - изменение 1 и 2 после заданного количества шагов генератора. В генератор можно добавть генерацию групп (промежуток появления существ для группы маленький, между группами - больше, для мини-передышки).
	- Минимум монет открывающих следующий уровень.
	- Набор существ и вероятность их появления.

СОЗДАНИЕ НОВЫХ НОР И ПОЯВЛЕНИЕ СУЩЕСТВ:
	- Одни существа могут как создавать новые норы, так и появляться из уже созданных. Другие - создавать не могут, только появляться в уже созданных.
	- Нору можно закопать или зацементировать. Если закопали, то в ее области все равно может появиться новая нора. Если зацементировали, то некоторое время в этой области не будут появляться новые норы.
	- Диаметр норы является множителем для способностей существ появляющихся из нор.
